Projeto 01-  Calculadora de Rank

Projeto básico em python para calcular o nivel de um jogador com base em vitorias e derrotas

Regras
Menos de 10 vitorias -> Ferro
11 a 20 -> Bronze
21 a 50 -> Prata
51 a 80 -> Ouro
81 a 90 -> Diamante
91 a 100 -> Lendario
101+ -> Imortal


Projeto 02 – Simulador de Combate

Pequeno projeto em Python criado para praticar programação orientada a objetos.

O código cria aventureiros de diferentes classes (arcanista, guardião, monge e sombrio) e simula ataques entre eles, causando dano aleatório até que um dos personagens seja derrotado.

Conceitos praticados:
- classes
- atributos e métodos
- instância de objetos
- lógica de combate
- uso da biblioteca random
